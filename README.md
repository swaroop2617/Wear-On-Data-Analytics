# WearOn Data Analysis

This project analyzes e-commerce order data exported from MongoDB to understand sales trends and business insights.
you can see more analytics in [http://wear-on-admin.vercel.app/dashboard](https://wear-on-admin.vercel.app/dashboard)

## Features

* Data cleaning using Python (pandas)
* Monthly and daily sales analysis
* Payment method and order status analysis
* Top product identification
* Basic business metrics (revenue, orders, average order value)

## Tech Stack

* Python
* Pandas
* Matplotlib

## Output

* Cleaned dataset (`orders_clean.csv`)
* Visual charts (stored in `charts/` folder)

## How to Run

pip install pandas matplotlib
python analyze_data.py

## Summary

This project demonstrates handling raw data, cleaning it, and extracting useful insights for decision-making in an e-commerce system.
